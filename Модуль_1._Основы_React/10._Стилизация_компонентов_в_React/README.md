# Стилизация компонентов в React

Интерактивный учебный проект для темы `Модуль 1 / 10. Стилизация компонентов в React`.

## Что покрывает тема

- `className` и обычные CSS-файлы;
- CSS Modules и изоляцию стилей;
- `style={{...}}` и dynamic runtime-значения;
- conditional styling и влияние состояния на внешний вид;
- переключение тем через CSS variables и `data-theme`;
- архитектуру стилизации компонентов и сравнение подходов;
- типичные ошибки при смешивании логики и стилей.

## Что внутри

Проект построен как одна учебная страница:

- сверху общий вводный блок по теме;
- под ним меню лабораторий;
- в центре активная лаборатория;
- снизу общий блок с тем, как читать проект, и стеком с версиями.

Внутри страницы 6 лабораторий:

1. `className и CSS`
   Показывает обычный CSS-файл, модификаторы и то, как один компонент меняет внешний вид через `className`.
2. `CSS Modules`
   Сравнивает глобальный CSS и module-изоляцию на живом примере с одноимёнными классами.
3. `Inline styles`
   Разбирает runtime-значения, `style={{...}}` и границы такого подхода.
4. `Conditional styling`
   Показывает, как состояние меняет внешний вид через variant maps и короткие модификаторы.
5. `Темы и токены`
   Переключает темы через CSS variables и контейнерный `data-theme`.
6. `Архитектура`
   Сравнивает подходы по ограничениям задачи и показывает более чистый style recipe.

На каждой лаборатории есть:

- живые переключатели и параметры;
- ссылки на файлы текущего проекта;
- листинги из этих файлов;
- короткие пояснения о практических последствиях выбранного подхода.

Важный принцип проекта: тема раскрывается не только в тексте, но и в коде самого приложения. Поэтому здесь есть реальные CSS-файлы, `.module.css`, `style={{...}}`, theme tokens, variant maps и компоненты, где стилизация организована так же, как она объясняется в интерфейсе.
