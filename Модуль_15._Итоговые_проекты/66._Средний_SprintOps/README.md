# SprintOps

Средний итоговый проект про `CRUD`, `dashboard`, `router`, `server state`, `auth flow`,
`protected routes`, `loading/error states`, `mutations`, `optimistic updates` и тесты.

Демо-логин:

- `lead@sprintops.io` / `react-ops`
- `ops@sprintops.io` / `react-ops`

Скрипты:

- `npm install`
- `npm run dev`
- `npm run build`
- `npm run test`
