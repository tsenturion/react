# Курс: React

Этот репозиторий — практический курс по современному React: от базы платформы, JSX, компонентов и состояния до маршрутизации, работы с данными, тестирования, производительности, React 19, full-stack-подхода, TypeScript и поддержки legacy-кода.

Здесь собраны учебные проекты и темы, которые последовательно раскрывают React не как набор разрозненных API, а как цельную систему проектирования интерфейсов, архитектуры приложения, пользовательского опыта, отладки, оптимизации и миграционной дисциплины.
